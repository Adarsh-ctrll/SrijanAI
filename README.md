# 🚀 SrijanAI – AI-Powered Website Generator

SrijanAI is an AI-powered platform that transforms natural language prompts into modern, responsive websites. Users can describe their ideas in plain English, generate production-ready code, preview the result instantly, edit it using an integrated code editor, and deploy their websites—all from a single interface.

🌐 **Live Demo:** https://srijanai-1.onrender.com/

---

## ✨ Features

- 🤖 AI-powered website generation from natural language prompts
- 💬 Interactive AI chat assistant
- ⚡ Live website preview
- 📝 Built-in Monaco code editor

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Adarsh-ctrll/SrijanAI.git
cd SrijanAI
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
OPENROUTER_API_KEY=your_api_key
MONGODB_URI=your_mongodb_uri
FIREBASE_CONFIG=your_firebase_config
STRIPE_SECRET_KEY=your_stripe_key
```

### Start the Application

```bash
npm run dev
```
